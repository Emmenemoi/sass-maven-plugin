2.5 Release Notes

A complete list of issues can be seen in the tracker milestone 2.5 .

This release features Compass version 1.0.3 and Sass version 3.4.13 running on JRuby 1.7.19. Please check the full dependencies to find out about the specific Sass and other transitive dependencies included in this release.

New

  • Add an optional parameter to specify the compass.rb file

Enhancements

  • Update maven libraries to 3.3.1 and start using Maven 3.3.1 in builds
  • Update scss-lint to version 0.37.0
