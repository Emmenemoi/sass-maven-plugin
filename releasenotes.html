2.10 Release Notes

A complete list of issues can be seen in the tracker milestone 2.10 .

This release features Compass version 1.0.3 and Sass version 3.4.16 running on JRuby 9.0.0.0. Please check the full dependencies to find out about the specific Sass and other transitive dependencies included in this release.

New

  • A skip parameter was added to skip plugin execution (#60)
  • Skip unmodified files (#18)

A complete list of issues can be seen in the tracker milestone 2.9 .

This release features Compass version 1.0.3 and Sass version 3.4.16 running on JRuby 9.0.0.0.

Fix

  • As a result of updating JRuby most Windows issues are expected to be fixed (#41),
