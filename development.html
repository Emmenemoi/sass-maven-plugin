Building from source

  • Check out the sourcecode 
    git checkout https://github.com/GeoDienstenCentrum/sass-maven-plugin sass-maven-plugin
  • build using Maven 
    cd sass-maven-plugin
mvn install

Please read the contributing document befor submitting a pull request. You may be required to sign the Contributor License Agreement before your contribution is merged.

Creating Release

Asuumeing the requirements for pushing to the OSRRH repository are set up, eg. account, gpg signature. use the following commands:

mvn clean release:clean release:prepare
git push --tags
mvn release:perform
git push

Then deploy the site to github:

mvn site-deploy -Dgithub.username=<username> -Dgithub.password=<password>
